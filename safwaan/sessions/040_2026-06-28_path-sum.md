# Session: Path Sum (LC 112) — 2026-06-28

## What He Attempted

**Iterative DFS first (cold, no hints):**
```js
function hasPathSum(root, targetSum) {
    if(root === null) return false;
    const stack = [[root, root.val]];
    while(stack.length > 0) {
        const [currentNode, currentSum] = stack.pop()
        if(currentNode.left === null && currentNode.right === null) {
            if(currentSum === targetSum) return true;
        }
        if(currentNode.right !== null) stack.push([currentNode.right, currentSum + currentNode.right.val])
        if(currentNode.left !== null) stack.push([currentNode.left, currentSum + currentNode.left.val])
    }
    return false;
};
```

**Recursive (guided):**
```js
function hasPathSum(root, targetSum) {
    if(root === null) return false;
    if(root.left === null && root.right === null) return targetSum === root.val;
    const leftHasTarget = hasPathSum(root.left, targetSum - root.val);
    const rightHasTarget = hasPathSum(root.right, targetSum - root.val);
    return leftHasTarget || rightHasTarget;
};
```

## Where He Got Stuck

- Recursive version: knew null → false but didn't know how to handle the running sum without an explicit variable
- Unlocked by: "what if you subtract the current node's value from targetSum as you pass it down?"

## Mistakes Made

- None in the iterative (direct transfer from fundamentals)
- Recursive needed the subtract-as-you-go insight explained

## Key Insight

Two different mental models for the same problem:
- **Iterative:** accumulate sum upward — push `[node, sum]` pairs, check `sum === targetSum` at leaf
- **Recursive:** subtract as you go down — pass `targetSum - root.val` into children. At the leaf, `targetSum === root.val` means the whole path balanced out
